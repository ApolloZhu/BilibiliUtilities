保证没有任何病毒，绝对没有，都是官方的。

1.下载 python:

https://www.python.org/ftp/python/3.5.1/python-3.5.1.exe


2.双击运行，注意：

勾选 Add Python 3.5 to PATH
然后点击 Install Now


3.按 windows徽标键+R，打开运行  / 或者是搜索 运行
在弹出的窗口输入

pip install requests


4.把`安装.py`移动到合适的文件夹，双击运行

5.向`UP主列表.txt`添加UP主的名字，输入完成之后一定要换行

6.通过更改`设置.txt`里面的内容可以做出一些小调整。
在编辑段位的时候，要遵守以下规律(除了最后一个之外都要有逗号，括号需要使用英文):

    "段位": [
        ["段位名", 数值],
        ["段位名", 数值],
        ["段位名", 数值],
        ...
        ["段位名", 数值]
    ]
